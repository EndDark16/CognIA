# Email assets

Place email images here (logos, signatures, icons).

Suggested files:
- static/email/logo/cognia-logo-light.png
- static/email/logo/cognia-logo-dark.png
- static/email/logo/cognia-signature.png
- static/email/icons/lock.svg
- static/email/icons/report.svg
- static/email/content/welcome-hero.png

Best practices:
- Keep each image under ~100KB
- Use PNG for logos/signatures in emails (SVG often fails on Gmail mobile/Outlook)
- Prefer PNG for complex illustrations or when Outlook compatibility is required
- Provide alt text in templates
- Host these assets on a public HTTPS URL and set EMAIL_ASSET_BASE_URL
